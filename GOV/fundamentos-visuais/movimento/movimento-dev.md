## Utilitários CSS de Movimento

São classes CSS para aplicar o Fundamento Visual Movimento.

### Como usar

Modifique a **atenuação** (*easing*) e a **duração** das animações.

> **Atenção!** As classes foram criadas para modificar o CSS *animation*. Não funcionarão com CSS *transition*.

#### Atenuação (*Easing*)

Controle a aceleração e desaceleração do movimento.

*Easing* define como o movimento acelera e desacelera ao longo do tempo, determinando se o movimento parece natural ou artificial.

Use as classes de atenuação disponíveis: `ease`, `ease-in`, `ease-out`, `ease-in-out`, `linear`

#### Duração

Controle o tempo de duração do movimento.

Use as classes de duração disponíveis: `very-fast`, `fast`, `moderate`, `slow`, `very-slow`
